REPLACEMENT INSTRUCTIONS:
-------------------------

1. GALLERY IMAGES:
   - Replace numbered images (1.jpg, 2.jpg, etc.) with actual school photos

   - Formats: JPG, PNG, WebP
   - Content ideas: classrooms, library, sports, events, students, facilities

2. TIMETABLE:
   - Replace sample-timetable.pdf with your current school timetable
   - Update the link in academics.js if filename changes

3. ADDITIONAL ASSETS:
   - School logo (if not using text-based logo)
   - Principal's photo for About page
   - School building photos
   - Event photos and videos

4. CONTENT UPDATES:
   - Update school name from "Blue Ridge School" to your actual name
   - Replace contact information in Footer and Contact components
   - Update address, phone, and email in all components
   - Customize colors in tailwind.config.js if needed

TECHNICAL NOTES:
----------------

- Use descriptive filenames for better SEO
- Keep file sizes reasonable (under 2MB for images)
